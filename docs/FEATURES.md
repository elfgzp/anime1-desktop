# Anime1 Desktop 产品功能介绍

从 [Anime1.me](https://anime1.me) 浏览和观看番剧的桌面客户端，支持 Windows、macOS 和 Linux。

## 首页浏览

首页展示最新番剧列表，支持分页浏览和快速访问。

![首页](./screenshots/home-page.png)

**功能特点：**
- 瀑布流布局展示番剧封面
- 显示番剧名称、最新集数、播出季度
- 分页导航，支持快速跳转
- 响应式设计，适配不同窗口大小

---

## 番剧详情

点击番剧可查看详细信息，包括简介、制作信息、剧集列表等。

![番剧详情](./screenshots/anime-detail.png)

**功能特点：**
- 番剧基本信息（年份、季度、评分、排名）
- 详细剧情简介
- 制作人员信息（原作、导演、脚本等）
- 剧集列表，可直接播放
- 关联 Bangumi 链接

## 主题模式

支持亮色和暗黑两种主题模式，保护您的眼睛。

### 亮色模式

![亮色模式设置](./screenshots/settings-light.png)

### 暗黑模式

![暗黑模式设置](./screenshots/settings-dark.png)

**功能特点：**

## 设置页面

提供应用配置和关于信息。

![设置页面](./screenshots/settings-light.png)

**功能特点：**
- 外观设置（主题模式）

## 我的追番

收藏喜欢的番剧，方便快速访问。

![我的追番](./screenshots/favorites-empty.png)

## 观看历史

记录观看历史，不再错过任何一集。

![观看历史](./screenshots/history-empty.png)

**功能特点：**
- 自动记录观看进度
- 快速续播

---

## 技术栈

- **前端**：Vue 3 + Vite
- **后端**：Flask
- **桌面框架**：Tauri
- **样式**：Tailwind CSS
