# Anime1 Desktop

Anime1 的桌面应用，跨平台支持 Windows / macOS / Linux。

从 [Anime1.me](https://anime1.me) 浏览和观看番剧的桌面客户端。

## 使用

直接运行对应平台的应用即可：
- macOS: 双击 `Anime1.app`
- Windows: 运行 `anime1.exe`
- Linux: 运行 `./anime1`

## 开发

```bash
# 安装依赖
make install

# 运行应用
make dev

# 打包
make build
```

详细开发说明请查看 [DEVELOPMENT.md](DEVELOPMENT.md)
