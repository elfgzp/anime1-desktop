## 下载

从 [GitHub Releases](https://github.com/elfgzp/anime1-desktop/releases) 下载最新版本。

### 支持平台

- **Windows (x64)**: `anime1-windows-x64.zip`
- **macOS (Intel)**: `anime1-macos-x64.dmg`
- **macOS (Apple Silicon)**: `anime1-macos-arm64.dmg`
- **Linux (x64)**: `anime1-linux-x64.tar.gz`
- **Linux (ARM64)**: `anime1-linux-arm64.tar.gz`

## 安装

### Windows

1. 下载 `anime1-windows-x64.zip`
2. 解压 zip 文件
3. 运行 `Anime1.exe`

### macOS

1. 根据你的 Mac 架构选择对应的版本：
   - Intel Mac: 下载 `anime1-macos-x64.dmg`
   - Apple Silicon (M1/M2/M3): 下载 `anime1-macos-arm64.dmg`
2. 双击 DMG 文件挂载
3. 将 `Anime1.app` 拖拽到应用程序文件夹
4. 在启动台或应用程序文件夹中找到并运行 Anime1

### Linux

1. 根据你的系统架构选择对应的版本：
   - x64: 下载 `anime1-linux-x64.tar.gz`
   - ARM64: 下载 `anime1-linux-arm64.tar.gz`
2. 解压文件：
   ```bash
   tar -xzf anime1-linux-*.tar.gz
   ```
3. 添加执行权限并运行：
   ```bash
   chmod +x Anime1
   ./Anime1
   ```
