# Homebrew 安装说明

## 从本地安装

```bash
# 进入项目目录
cd anime1-desktop

# 安装 (自动匹配架构)
brew install --cask ./homebrew/anime1.rb
```

## 从 DMG 安装

```bash
# 下载 DMG 后安装
brew install --cask ~/Downloads/anime1-macos-arm64.dmg
```

## DMG 安装方式

1. 双击挂载 DMG
2. 你会看到两个图标：
   - **Anime1.app** - 拖拽到应用程序文件夹
   - **修复签名** - 双击运行修复签名问题
3. 安装完成后运行 Anime1
